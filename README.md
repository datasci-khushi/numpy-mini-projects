# NumPy Mini Projects

Welcome to my NumPy mini-project repository.

This repository contains beginner-level projects that I built while learning NumPy as part of my Data Science journey. These projects helped me strengthen my understanding of arrays, data filtering, aggregation, basic calculations, and random number generation through hands-on practice.

## 📂 Projects

### 1. Student Marks Analysis

Analyzed student marks data using NumPy arrays and performed:

* Average marks calculation
* Highest and lowest marks identification
* Pass/fail analysis
* Pass percentage calculation
* Grace marks calculation

### 2. Employee Salary Analysis

Analyzed employee salary data using NumPy operations and performed:

* Average salary calculation
* Highest and lowest salary identification
* Salary filtering based on conditions
* Salary increment analysis
* Total salary calculation

### 3. Dice Simulator

Simulated 100 dice rolls using NumPy's random module and performed:

* Counting occurrences of specific dice results
* Average roll calculation
* Maximum and minimum roll identification
* Conditional outcome analysis

### 4. Sales Analysis

Analyzed sales data across multiple products and months and performed:

* Product-wise sales analysis
* Month-wise sales analysis
* Best-performing product identification
* Best-performing month identification
* Average, highest, and lowest sales calculation

## 🧠 NumPy Concepts Practiced

* Creating NumPy Arrays
* Array Operations
* Boolean Filtering
* Aggregation Functions
* Random Number Generation
* Working with 1D and 2D Arrays

## 🎯 Learning Objective

These projects represent my practical learning journey with NumPy and provide a foundation for further learning in Pandas, Data Visualization, Statistics, and Machine Learning.
